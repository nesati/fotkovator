# Fotkovátor - Technická dokumentace

## Přehled systému
### Core

- čte konfiguraci
- spouští moduly

### Moduly

#### Blackend

- hlavní proces
- přistupuje k fotkám a obrázkům
- získává metadata fotek
- může být jen jeden

#### Štítky

- přidává štítky fotkám

#### Databáze

- ukládá dlouhodobě data

#### Frontend

- umožňuje přístup uživateli
- může být jen jedna
